Screencast: The Use of Google Read and Write

For this project, I screencast a video where I reviewed some of the features accessible to users with the Google Read and Write Chrome extension. I recorded my screencast using Camtasia. Before recording, I set up the Zoom features of my document so it could be easily viewed. I then changed my cursor type, the scale of my mouse pointer and added a cursor glow to the mouse with a change to the cursor color. To help with attention, I added a yellow circle callout to the tools as I was reviewing them within the video. I then exported the video to an mp4 and then uploaded the video to Youtube.
