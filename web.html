Web Design

Javascript and Drag&Drop Quizzes (Click on each image below to open up the activity)

For the JavaScript quiz, I created a 10-question quiz based on Disney and Pixar movies.


Structure: With this project, I learned to alter Javascript code. I modified quiz questions so that there was a total of 10 questions. I modified each question to be based on popular Disney and Pixar movies and characters. I also updated the responses to the questions, changed the code connected to the correct answers, and then updated the solutions. For the solutions, I changed the original ones to include famous movie quotes, when the correct answer was selected (e.g. Excellent! Do not just fly, soar.; Wow, you're really rocking! Believe you can, then you will.; Great Job! To infinity and beyond! etc.) I also learned how to add images to a quiz and create a link to the image.



For the Drag&Drop quiz, I deconstructed and modified a script for a drag-n-drop assessment about recycling for kids.


Structure: With this project, I continued to practice altering javascript code. I also learned new techniques with CSS such as changing the color of text, adding and defining images, and playing around with images and hyperlinks.
